# Delight Audit — Where the Illusion Breaks

**Lens:** Does this feel like returning to a warm cabin after a day in the mountains?

## Still generic or rushed

| Area | Issue | Severity |
|------|--------|----------|
| Parallax layers | Same image stacked without true depth masks — motion reads as placeholder | High |
| Sample portfolio | SVG placeholders instead of real field photographs | High |
| Analyzer suggestions | Helpful but still algorithm-shaped; needs softer “naturalist glance” framing | Medium |
| Coming soon grid | Long roadmap of AI features — reads as product backlog, not studio journal | Medium |
| Tab bar (mobile) | Five studios on one row — cramped on small phones | Medium |
| Export | Single PNG frame only — emotional arc stops at “share” | Medium |
| File picker | Native OS dialog — unavoidable, breaks cabin immersion briefly | Low |

## Still feels like a coding project

| Area | Issue |
|------|--------|
| Percentage sliders everywhere | Engineer affordance; Lightroom hides complexity better |
| `data-tab` / `tab-panel` IDs | Internal names (`living-scene`, `parallax`) diverge from studio labels |
| Console stubs in `export.js` | Developer residue |
| Duplicate CSS layers (`main.css` + `studio-shell.css`) | Maintenance smell, occasional visual drift |
| No real asset pipeline | `assets/Images/` unused — portfolio not wired to real work |

## Premium illusion breaks

| Moment | Why |
|--------|-----|
| First import | Works — hero copy now supports homecoming |
| Empty easel | Improved — “Set a frame on the easel” |
| Preset names | Strong — weather as mood |
| Collections modal | Strong — field notes, captions, warm metadata |
| Field Guide | Improved — exercises, reflection, challenges |
| Parallax without masks | Weak — ghost layers hidden; depth is subtle not magical |
| “On the horizon” cards | Acceptable if quiet; still a feature list |

## Delight that already lands

- Homecoming hero: *You made it home. Let the day breathe.*
- Unpack / hearth / trail vocabulary
- Living Scene atmosphere presets as moods
- Collections as wall by the fire
- Field Guide as post-sunset practice
- Local-only privacy — trust for photographers
- Reduced camera defaults — observation not drone

## QA checklist (emotional)

- [ ] Open app cold — do you exhale?
- [ ] Unpack a frame — does it feel like unzipping a bag, not uploading a file?
- [ ] Living Scene — does motion feel like memory, not demo?
- [ ] Parallax — does it feel like leaning at a window?
- [ ] Collections — do you want to linger?
- [ ] Field Guide — do you want to go outside tomorrow?
- [ ] Export — does saving feel like sharing one breath?
